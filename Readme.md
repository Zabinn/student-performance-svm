# Student Performance Analysis & Prediction (SVM)

## Objective
Identify the key factors influencing student exam performance, and build a
classification model to predict whether a student will score above or below
the median exam score.

## Dataset
Student Performance Factors dataset (study hours, attendance, parental
involvement, access to resources, extracurricular activities, exam scores).

## Plan
1. Data cleaning (missing values, categorical standardization, score range checks)
2. Exploratory Data Analysis (distributions, scatterplots, boxplots)
3. Relationship analysis (study hours, parental involvement, resources, etc. vs score)
4. Outlier detection
5. Binary target creation (median split: high performer / needs support)
6. SVM (RBF kernel) classification model
7. Evaluation (accuracy, confusion matrix, recall)
8. Recommendations

## Tech Stack
Python, pandas, matplotlib/seaborn, scikit-learn, Jupyter
